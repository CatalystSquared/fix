Reference Architecture

FIX is a sample responsive web application that demonstrates how to build ecommerce systems with Salesforce and open-source frameworks like Ruby on Rails.

Track all your customer information and interactions in one place. The #1 CRM solution takes sales and support automation to new heights, with the power to grow with your business.

FIX uses Salesforce as the system of record for customer support agents.

Salesforce website »

Heroku makes the processes of deploying, configuring, scaling, tuning, and managing apps as simple and straightforward as possible, so developers can focus on what’s most important: building great apps.

FIX uses Heroku's cloud platform to run the Ruby on Rails app and all of its supporting services.

Heroku website »

The Spree storefront offers a full feature set and is built on common standards. The modular platform allows you to easily configure, supplement or replace any functionality you need, so that you can build the exact storefront that you want.

FIX uses a customized version of Spree to produce the eCommerce experience and processes.

Spree website »

Using bi-directional synchronization between Salesforce and Heroku Postgres, Heroku Connect unifies the data in your Postgres database with the contacts, accounts, or any type of object in the Salesforce database.

FIX uses Connect to instantly reflect customers and their orders as Salesforce records.

Connect website »

Ruby on Rails is an open-source web framework that’s optimized for programmer happiness and sustainable productivity. It lets you write beautiful code by favoring convention over configuration.

FIX uses Rails' modularity to enable extensive customization of the Spree storefront.

Rails website »

HTML, CSS, and JS framework for developing responsive, mobile first projects on the web. Bootstrap makes front-end web development faster and easier. It's made for folks of all skill levels, devices of all shapes, and projects of all sizes.

FIX uses Bootstrap to create a beautiful interface for any modern device, from phones to tablets to desktop PCs.
